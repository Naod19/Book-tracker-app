# Book Tracker App 📕

A personal reading list tracker built with vanilla JavaScript, HTML and CSS.

## Features
- Add books with title, author, date and reading status
- Search books by title or author
- Filter books by reading status (Read, Not Started, Favorite)
- Delete books
- Data persists with localStorage

## What I Learned
- Dynamic DOM manipulation
- localStorage for data persistence
- Filtering and searching arrays using JavaScript array methods
- CSS Grid for responsive card layout
- Modal/popup implementation with CSS transitions
- Mobile-first responsive design using clamp() and viewport units

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Use
1. Click "Add Book" to add a new book
2. Fill in the title, author, date and reading status
3. Use the search bar to find books by title or author
4. Use the filter dropdown to filter by reading status
5. Click "Delete Book" to remove a book
